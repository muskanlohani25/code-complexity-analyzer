# Security Policy

## Supported Versions

We actively support and patch security issues in the following versions:

| Version | Supported |
| ------- | --------- |
| 1.2.x   | Yes       |
| 1.1.x   | No        |
| 1.0.x   | No        |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability or security issue, please do not report it in a public issue tracker.

Instead, please email the details to the maintainer email. We will investigate the issue and attempt to coordinate a fix before disclosing it publicly.
